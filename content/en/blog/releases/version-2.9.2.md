## Changes from 2.9.1 to 2.9.2dev1

There are only UI-updates and not model changes, the build should be as stable as the released version. There will be updates to the build before next release.

- Removed Verify Tasks UI and cleaned up the sidebar
- Added Verify Task tab and renamed DefaultView
- Moved toolbar to navigation and cleared title
- Moved LogRecordsTabView; added conditional searchable
- Showed clear toolbar only on Log view
- Removed Logs-by-configuration view and references
- Added LogRecords tab and fixed help text formatting
- Simplified and reworded help text strings
